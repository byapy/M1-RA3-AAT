# M1RA3AAT Sitio web Intecap

![Intecap-Logo](https://intecap.edu.gt/wp-content/uploads/2021/06/logo-intecap-760x336.png)
### Descripción
Esta es una pagina web pensada para INTECAP. En este sitio puedes acceder a la información general del instituto, registrar empleados en tablas y gestionarlos. Calculadora de sueldo liquido mediante Ingresos y Egresos, se calcula IGSS y posibles prestamos. Por ultimo podemos averiguar cuanto nos toca de indemnización según el sueldo base y los años que llevamos trabajando.
#### Tecnologías Utilizadas:
| HTML | CSS |  JavaScript  | Bootstrap
|--|--|--|--|
| Para formar la estructura | CSS externo para personalización | Scripts para funcionalidad general de formularios  | Framework utilizado para diseños rápidos y genéricos

- Identidad visual clara
- Header con logo y redes sociales
